# 认识markdown 
***
markdown是一种用来写作的轻量级语言，使用简洁的语法来代替码字
# markdown的标题
***
'#'表示的是从一个到多个一次表示的各种标题的降级，在#号之后建议使用空格号进行隔开这个也是markdown的语法格式
#markdown的列表
***
列表又可以分为有序列表和无序列表两种，列表的显示直接前缀加上-或者*即可，无序列表加上（1234）.的方式即可
### 有序列表例子1
- 列1 
- 列2
- 列3
### 无序列表例子2
1. 例1
2. 例2
3. 例3
